# VibeEdit 0.1.0 beta 1

This public beta is for evaluation, testing, and community review. It exercises
the complete VibeEdit package across Python, Node, deterministic HTML motion,
FFmpeg rendering, vision adapters, optional SAM 2.1, the static catalog, MCP,
and four AI-harness skill layouts.

## Install

Python 3.11 or newer:

```bash
uv tool install https://github.com/Momenta-App/vibeedit-sdk/releases/download/v0.1.0-beta.1/vibeedit-0.1.0b1-py3-none-any.whl
vibeedit doctor --json
```

Node 20 or newer:

```bash
npm install https://github.com/Momenta-App/vibeedit-sdk/releases/download/v0.1.0-beta.1/vibeedit-0.1.0-beta.1.tgz
npx vibeedit doctor --json
```

The release also includes a Python source distribution, SHA-256 sums, and an
archive audit. Verify GitHub build provenance with:

```bash
gh attestation verify vibeedit-0.1.0b1-py3-none-any.whl --repo Momenta-App/vibeedit-sdk
```

## Known beta boundaries

- SAM 3.1 remains quarantined and is not claimed.
- Pose detection is macOS-native in this release.
- ONNX Runtime does not publish a macOS Intel wheel for the pinned version, so
  the portable object provider reports a structured unsupported state there.
- Browser, vision, and SAM runtimes are optional explicit downloads.
- The beta is not a production-stability or commercial-use grant. See
  `LICENSE.md` and `THIRD_PARTY_NOTICES.md`.

All 13 packaged preview/audio assets are generated by VibeEdit and hash-bound.
No movie clips, music, dialogue, images, or other third-party recorded media are
included. Canonical skills and preset sources remain byte-identical to their
pinned VibeEdit revision.
